# App Icon

To add a custom icon:

1. Create or download a 256x256 PNG icon
2. Convert it to ICO format using: https://convertio.co/png-ico/
3. Save as `icon.ico` in this folder
4. Rebuild the app with `npm run build`

For now, Electron will use the default icon.

You can also use these free icon resources:
- https://icons8.com
- https://www.flaticon.com
- https://www.iconfinder.com
