# 👤 User Guide - How to Use AppLauncher

## 📥 First Time Installation

### Step 1: Download
You will receive: **`AppLauncher-Setup.exe`**

### Step 2: Install (ONE TIME ONLY)
1. Double-click `AppLauncher-Setup.exe`
2. Choose where to install (or use default location)
3. Click "Install"
4. App opens automatically after installation ✅

### Step 3: Done!
- App is now installed on your computer
- Desktop shortcut created (optional)
- Start Menu shortcut created

---

## 🚀 Daily Usage

### How to Open the App:

**Option 1:** Click the desktop shortcut (if you created one)

**Option 2:** Start Menu → Search "AppLauncher" → Click it

**Option 3:** Look for the small **purple line** on the left edge of your screen

---

## 🎯 How the App Works

### The Purple Line:
- You'll see a small **vertical purple line** on the left edge of your screen
- This is the app running in the background

### To Use It:
1. **Move your mouse to the left edge** of the screen
2. The purple line **expands into a sidebar** showing your apps
3. **Click any app** to open it in your browser
4. **Move mouse away** → Sidebar collapses back to purple line

### Visual Guide:
```
Normal:                   Mouse near edge:
┃ ← Purple line          ┏━━━━━━━┓
┃                        ┃ 🌐 App1 ┃
┃                        ┃ 📧 App2 ┃
┃                        ┃ 💼 App3 ┃
                         ┗━━━━━━━┛
```

## ❓ Common Questions

### Q: Do I run Setup.exe every time?
**A: NO!** Setup.exe is ONLY for installation.

- **First time:** Run Setup.exe to install ✅
- **After that:** Use Start Menu or desktop shortcut ✅
- **Never run Setup.exe again** unless reinstalling ❌

### Q: Where did the app go after I closed it?
**A: It's always running!**

- Look for the **purple line** on the left edge of your screen
- The app doesn't appear in the taskbar (by design)
- It's a "sidebar" that shows when you need it

### Q: How do I close the app completely?
1. Move mouse to left edge to expand sidebar
2. Click the **red ×** button at the top
3. App closes completely

### Q: How do I restart the app?
- Start Menu → "AppLauncher"
- Or click desktop shortcut

### Q: Will I get updates automatically?
**A: YES!**

- App checks for updates automatically
- Downloads and installs in the background
- You don't need to do anything
- Just restart the app occasionally

## 🔧 Troubleshooting

### "I don't see the purple line!"
- App might not be running
- Open it from Start Menu → "AppLauncher"
- Check if it's running in Task Manager

### "Sidebar doesn't expand when I move mouse"
- Make sure you move mouse **all the way to the left edge** (0 pixels from edge)
- Try closing and reopening the app

### "I accidentally deleted the desktop shortcut"
- Don't worry! It's still installed
- Find it in: Start Menu → "AppLauncher"
- Or reinstall to recreate shortcuts

### "How do I uninstall?"
- Windows Settings → Apps → Find "AppLauncher"
- Click Uninstall
- Or run the setup again and choose "Uninstall"
